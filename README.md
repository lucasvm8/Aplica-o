# Aplica

1. Clone the repo
   ```sh
   git clone https://github.com/lucasvm8/Projeto-Faculdade
   ```
2. Install NPM packages
   ```sh
   npm i express --save
   ```
   ```sh
   npm i pdf-parse
   ```
   ```sh
   npm i ejs
   ```
   ```sh
   npm i multer
   ```
   ```sh
   npm i cors
   ```
   ```sh
   npm i node-fetch
   ```
3. go to http://localhost:8080
